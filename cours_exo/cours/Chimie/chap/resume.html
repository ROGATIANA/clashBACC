🧪 PARTIE 1 : CHIMIE ORGANIQUE

📌 Les Alcools

Formule générale

R-OH

Oxydation (alcool primaire)

R-CH₂OH → R-CHO → R-COOH

Oxydation (alcool secondaire)

R-CHOH-R' → R-CO-R'

Déshydratation

R-CH₂-CH₂OH → R-CH=CH₂ + H₂O

📌 Les Amines

Formule générale

R-NH₂ (primaire), R₂NH (secondaire), R₃N (tertiaire)

Basicité

RNH₂ + H₂O ⇌ RNH₃⁺ + OH⁻

Constante de basicité

K_b = [RNH₃⁺][OH⁻]/[RNH₂]

📌 Les Acides Carboxyliques et Dérivés

Acide carboxylique

R-COOH

Ester

R-COO-R'

Estérification

R-COOH + R'-OH ⇌ R-COO-R' + H₂O

Hydrolyse d'un ester

R-COO-R' + H₂O ⇌ R-COOH + R'-OH

📌 Cinétique Chimique

Vitesse de réaction

v = (1/V)·dξ/dt

Loi de vitesse (ordre 0)

v = k

Loi de vitesse (ordre 1)

v = k[A]

Loi d'Arrhénius

k = A·e^{-E_a/RT}

⚗️ PARTIE 2 : CHIMIE MINÉRALE

📌 pH d'une Solution Aqueuse

Produit ionique de l'eau

K_e = [H₃O⁺][OH⁻] = 10⁻¹⁴ à 25°C

pH

pH = -log[H₃O⁺]

pOH

pOH = -log[OH⁻]

Relation pH + pOH

pH + pOH = 14

📌 Acide Fort - Base Forte

Acide fort (C)

pH = -log C

Base forte (C)

pH = 14 + log C

Dosage acide fort/base forte

pH à l'équivalence = 7

📌 Acide Faible - Base Faible

Acide faible (C)

pH = ½(pK_a - log C)

Base faible (C)

pH = 7 + ½(pK_a + log C)

Constante d'acidité

K_a = [A⁻][H₃O⁺]/[AH]

pKa

pK_a = -log K_a

📌 Solutions Tampons

Équation de Henderson-Hasselbalch

pH = pK_a + log([base]/[acide])

Pouvoir tampon

β = dC_B/dpH

📌 Les Acides α-Aminés

Formule générale

H₂N-CHR-COOH

Point isoélectrique (pI)

pI = (pK_a1 + pK_a2)/2

Liaison peptidique

-CO-NH-

📌 CONSTANTES FONDAMENTALES

Produit ionique de l'eau

K_e = 1,0×10⁻¹⁴

Nombre d'Avogadro

N_A = 6,02×10²³ mol⁻¹

Constante des gaz parfaits

R = 8,314 J·mol⁻¹·K⁻¹

Volume molaire (CNTP)

V_m = 22,4 L·mol⁻¹

💡 CONSEILS POUR LE BACCALAURÉAT

📌 À retenir :
- Toujours vérifier l'équilibrage des équations chimiques
- Maîtriser la notation des groupes fonctionnels en chimie organique
- Connaître les relation pH/pOH/K_e et les formules de pH pour les différents cas
- Savoir utiliser l'équation de Henderson-Hasselbalch pour les tampons
- Vérifier les unités (concentration en mol/L, volume en L)
- Soigner la rédaction des mécanismes réactionnels
