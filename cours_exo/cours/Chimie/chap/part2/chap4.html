Constante d'acidité

Force des acides · Échelle de pKₐ · Classification · Diagramme de prédominance

Cours complet

📊 INTRODUCTION

La constante d'acidité Kₐ (ou son logarithme pKₐ) est une grandeur fondamentale qui permet de classifier les acides selon leur force. Plus Kₐ est grand (plus pKₐ est petit), plus l'acide est fort. Cette constante permet également de déterminer la forme prédominante d'un couple acide/base en fonction du pH.

📌 Notions clés :
- Kₐ : constante d'équilibre de dissociation d'un acide faible
- pKₐ : -log Kₐ (échelle de force acide)
- Échelle de pKₐ : de -10 (acides très forts) à 50 (acides très faibles)
- Diagramme de prédominance : espèce majoritaire selon le pH

📚 I – CONSTANTE D'ACIDITÉ Kₐ

Définition

Pour un couple acide/base AH / A⁻ :

AH + H₂O ⇌ A⁻ + H₃O⁺

La constante d'équilibre est : K_a = [A⁻][H₃O⁺] / [AH]

Kₐ est indépendante de la concentration et ne dépend que de la température.

pKₐ

pK_a = -log K_a

L'échelle de pKₐ permet de comparer facilement la force des acides :

  • pKₐ < 0 : acide fort (dissociation totale)
  • 0 < pKₐ < 14 : acide faible
  • pKₐ > 14 : acide très faible

📊 II – CLASSIFICATION DES COUPLES ACIDE/BASE

📌 Exemples de pKₐ :
- Acides forts : HCl (pKₐ ≈ -7), HNO₃ (pKₐ ≈ -1,4), H₂SO₄ (pKₐ₁ ≈ -3)
- Acides moyens : H₃PO₄ (pKₐ₁ = 2,1), HF (pKₐ = 3,2)
- Acides faibles : CH₃COOH (pKₐ = 4,76), H₂CO₃ (pKₐ₁ = 6,35)
- Acides très faibles : NH₄⁺ (pKₐ = 9,25), H₂O (pKₐ = 15,7)

Acides polyacides

Un acide polyacide (comme H₃PO₄, H₂CO₃) possède plusieurs constantes d'acidité :

K_{a1} > K_{a2} > K_{a3} (chaque acidité successive est plus faible)

Exemple pour H₃PO₄ :

  • pK_{a1} = 2,1 (H₃PO₄ ⇌ H₂PO₄⁻ + H⁺)
  • pK_{a2} = 7,2 (H₂PO₄⁻ ⇌ HPO₄²⁻ + H⁺)
  • pK_{a3} = 12,3 (HPO₄²⁻ ⇌ PO₄³⁻ + H⁺)

📐 III – DIAGRAMME DE PRÉDOMINANCE

Principe

Le diagramme de prédominance indique, pour un couple acide/base, quelle est l'espèce majoritaire en fonction du pH.

Relation : pH = pK_a + log([A⁻]/[AH]) (équation de Henderson-Hasselbalch)

Règles de prédominance

  • Si pH < pK_a : AH prédomine (la forme acide)
  • Si pH = pK_a : [AH] = [A⁻] (50% de chaque forme)
  • Si pH > pK_a : A⁻ prédomine (la forme basique)
📌 Représentation graphique :
[AH] > [A⁻]  |  [AH] = [A⁻]  |  [A⁻] > [AH]
   pH < pKₐ        pH = pKₐ        pH > pKₐ

⚡ IV – ÉCHELLE DE FORCE DES BASES

Relation avec pKₐ

La force d'une base B est liée à la force de son acide conjugué BH⁺ :

K_b = K_e / K_a et pK_b = 14 - pK_a

Plus l'acide conjugué est fort (pKₐ petit), plus la base est faible (pK_b grand).

Plus l'acide conjugué est faible (pKₐ grand), plus la base est forte (pK_b petit).

📌 Exemples :
- Cl⁻ (pKₐ = -7) : base très faible (ne réagit pas avec l'eau)
- CH₃COO⁻ (pKₐ = 4,76) : base faible (pK_b = 9,24)
- NH₃ (pKₐ = 9,25) : base moyenne (pK_b = 4,75)
- H₂O (pKₐ = 15,7) : base très faible (pK_b = -1,7)

📝 V – FORMULAIRE RÉCAPITULATIF

Constante d'acidité

K_a = [A⁻][H₃O⁺]/[AH]
pK_a = -log K_a

Henderson-Hasselbalch

pH = pK_a + log([A⁻]/[AH])

Relation acide/base

K_a·K_b = K_e = 10⁻¹⁴
pK_a + pK_b = 14

Prédominance

pH < pKₐ : AH prédomine
pH > pKₐ : A⁻ prédomine

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Kₐ mesure la force d'un acide : plus Kₐ est grand, plus l'acide est fort
- pKₐ = -log Kₐ : plus pKₐ est petit, plus l'acide est fort
- L'échelle de pKₐ permet de classifier les acides (forts, faibles, très faibles)
- Diagramme de prédominance : pH < pKₐ → AH majoritaire ; pH > pKₐ → A⁻ majoritaire
- Pour un acide polyacide : pK_{a1} < pK_{a2} < pK_{a3}
- pK_a + pK_b = 14 pour un couple acide/base conjugués
🎯 Méthode pour déterminer l'espèce prédominante :
1. Connaître le pKₐ du couple acide/base
2. Connaître le pH de la solution
3. Comparer pH et pKₐ :
   - Si pH < pKₐ → forme acide AH prédominante
   - Si pH > pKₐ → forme basique A⁻ prédominante
   - Si pH = pKₐ → [AH] = [A⁻]
