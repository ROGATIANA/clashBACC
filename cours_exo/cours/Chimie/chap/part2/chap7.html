📚 INTRODUCTION

Ce chapitre regroupe les documents annexes utiles pour la résolution des exercices de chimie : tables de constantes, formules essentielles, classification périodique simplifiée, indicateurs colorés, etc. Ces documents sont fournis lors des épreuves du Baccalauréat.

📌 Utilisation :
- À consulter régulièrement lors des révisions
- À utiliser lors des exercices pour retrouver rapidement une constante
- À maîtriser pour les épreuves du Baccalauréat

📝 V – FORMULES ESSENTIELLES

Concentration molaire

C = n / V

Quantité de matière

n = m / M = C·V = N / N_A = V_gaz / V_m

Dilution

C₀·V₀ = C·V

pH

pH = -log[H₃O⁺]

Acide fort

pH = -log C

Base forte

pH = 14 + log C

Acide faible

pH = ½(pK_a - log C)

Henderson-Hasselbalch

pH = pK_a + log([base]/[acide])
