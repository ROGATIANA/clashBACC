Acide faible - Base faible

Couple acide/base · Constante d'acidité Kₐ · Équilibre chimique · Calcul du pH

Cours complet

⚖️ INTRODUCTION

Un acide faible est un acide qui ne se dissocie que partiellement dans l'eau. Une base faible est une base qui ne se dissocie que partiellement dans l'eau. Contrairement aux acides/bases forts, la réaction de dissociation est un équilibre chimique caractérisé par une constante d'équilibre K (Kₐ pour les acides, K_b pour les bases).

📌 Exemples :
- Acides faibles : CH₃COOH (acide éthanoïque), H₂CO₃, H₃PO₄, HF
- Bases faibles : NH₃ (ammoniac), CH₃NH₂ (méthylamine), C₅H₅N (pyridine)

📚 I – DÉFINITIONS

Acide faible

AH + H₂O ⇌ A⁻ + H₃O⁺

La réaction est limitée : la majorité des molécules AH reste sous forme non dissociée.

Base faible

B + H₂O ⇌ BH⁺ + OH⁻

La réaction est limitée : la majorité des molécules B reste sous forme non protonée.

📊 II – CONSTANTE D'ACIDITÉ Kₐ ET pKₐ

Constante d'acidité

K_a = [A⁻][H₃O⁺] / [AH]

Kₐ est une constante qui ne dépend que de la température. Plus Kₐ est grand, plus l'acide est fort.

pKₐ

pK_a = -log K_a

Plus pKₐ est petit, plus l'acide est fort.
Plus pKₐ est grand, plus l'acide est faible.

📌 Exemples de pKₐ :
- Acide chlorhydrique (fort) : pKₐ < 0
- Acide éthanoïque : pKₐ = 4,76
- Acide carbonique : pKₐ₁ = 6,35 ; pKₐ₂ = 10,33
- Ammonium NH₄⁺ : pKₐ = 9,25

📐 III – CALCUL DU pH

Acide faible de concentration C

Formule approchée (si C ≫ [H₃O⁺] ≫ 10⁻⁷) :

[H₃O⁺] = √(K_a·C)

pH = ½(pK_a - log C)

Base faible de concentration C

On utilise la constante de basicité K_b = [BH⁺][OH⁻]/[B]

Avec K_b = K_e / K_a et pK_b = -log K_b

[OH⁻] = √(K_b·C)

pH = 7 + ½(pK_a + log C)

📌 Conditions de validité :
- C ≫ [H₃O⁺] pour un acide faible (taux de dissociation faible)
- [H₃O⁺] ≫ 10⁻⁷ mol/L (pas trop dilué)
- Pour les solutions très diluées, tenir compte de l'autoprotolyse de l'eau

🔗 IV – RELATION ENTRE Kₐ ET K_b

Pour un couple acide/base conjugués

K_a × K_b = K_e = 10^{-14} (à 25°C)

pK_a + pK_b = 14

📌 Conséquences :
- Plus l'acide est fort (pKₐ faible), plus sa base conjuguée est faible (pK_b grand)
- Plus l'acide est faible (pKₐ grand), plus sa base conjuguée est forte (pK_b faible)
- Relation fondamentale pour les calculs sur les bases faibles

📝 V – FORMULAIRE RÉCAPITULATIF

Constante d'acidité Kₐ

K_a = [A⁻][H₃O⁺]/[AH]
pK_a = -log K_a

Acide faible (C)

pH = ½(pK_a - log C)

Base faible (C)

pH = 7 + ½(pK_a + log C)

Relation Kₐ·K_b

K_a·K_b = K_e = 10⁻¹⁴
pK_a + pK_b = 14

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Un acide faible est partiellement dissocié (équilibre)
- Kₐ mesure la force d'un acide : plus Kₐ est grand, plus l'acide est fort
- pKₐ = -log Kₐ : plus pKₐ est petit, plus l'acide est fort
- Pour un acide faible de concentration C : pH = ½(pK_a - log C)
- Pour une base faible : pH = 7 + ½(pK_a + log C)
- K_a·K_b = K_e = 10^{-14} et pK_a + pK_b = 14
🎯 Méthode pour calculer le pH d'un acide faible :
1. Vérifier que C ≫ 10⁻⁷ mol/L
2. Appliquer la formule approchée pH = ½(pK_a - log C)
3. Vérifier que [H₃O⁺] = 10^{-pH} ≪ C (taux de dissociation < 5%)
4. Si la condition n'est pas vérifiée, résoudre l'équation du second degré
[H₃O⁺]² + K_a[H₃O⁺] - K_a·C = 0
