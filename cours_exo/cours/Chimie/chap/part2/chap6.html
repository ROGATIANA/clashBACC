Les acides α-aminés

Structure · Classification · Stéréochimie · Propriétés acido-basiques · Point isoélectrique

Cours complet

🧬 INTRODUCTION

Les acides α-aminés sont des molécules organiques possédant à la fois un groupe amine (-NH₂) et un groupe acide carboxylique (-COOH) sur le même atome de carbone (le carbone α). Ils sont les monomères constitutifs des protéines. Il existe 20 acides aminés protéinogènes standards.

📌 Formule générale : H₂N-CH(R)-COOH
- R : chaîne latérale (différente selon l'acide aminé)
- Le carbone α est asymétrique (sauf pour R = H, glycine)

📚 I – STRUCTURE GÉNÉRALE

Formule développée

H₂N-CH-COOH
         |
        R

Classification selon la chaîne latérale R

  • Apollaires (hydrophobes) : Glycine (Gly), Alanine (Ala), Valine (Val), Leucine (Leu), Isoleucine (Ile), Proline (Pro), Phénylalanine (Phe), Tryptophane (Trp), Méthionine (Met)
  • Polaires non chargés : Sérine (Ser), Thréonine (Thr), Cystéine (Cys), Tyrosine (Tyr), Asparagine (Asn), Glutamine (Gln)
  • Chargés positivement (basiques) : Lysine (Lys), Arginine (Arg), Histidine (His)
  • Chargés négativement (acides) : Acide aspartique (Asp), Acide glutamique (Glu)
📌 Acides aminés essentiels :
Ceux que l'organisme ne peut pas synthétiser : Histidine, Isoleucine, Leucine, Lysine, Méthionine, Phénylalanine, Thréonine, Tryptophane, Valine.

🔬 II – STÉRÉOCHIMIE DES ACIDES α-AMINÉS

Carbone asymétrique

Sauf pour la glycine (R = H), tous les acides aminés possèdent un carbone asymétrique et sont donc chiraux. Ils existent sous deux formes énantiomères (image l'une de l'autre).

Configuration L et D

  • Les acides aminés naturels sont tous de configuration L
  • Les acides aminés de configuration D sont rares dans la nature (présents dans certaines parois bactériennes)
  • Les êtres vivants ne possèdent que des acides aminés L (homochiralité)
📌 Règle : Un acide aminé est de configuration L si dans la projection de Fischer, le groupe -NH₂ est à gauche du carbone α.

⚖️ III – PROPRIÉTÉS ACIDO-BASIQUES

Caractère amphotère

Les acides aminés sont des ampholytes : ils possèdent à la fois un groupe acide (-COOH) et un groupe basique (-NH₂).

  • En milieu acide (pH bas) : H₃N⁺-CH(R)-COOH (cation)
  • Au point isoélectrique : H₃N⁺-CH(R)-COO⁻ (zwitterion)
  • En milieu basique (pH haut) : H₂N-CH(R)-COO⁻ (anion)

Point isoélectrique (pI)

Le point isoélectrique (pI) est le pH pour lequel la charge nette de l'acide aminé est nulle (forme zwitterionique majoritaire).

pI = (pK_{a1} + pK_{a2})/2 pour un acide aminé neutre

À ce pH, l'acide aminé est le moins soluble dans l'eau.

📌 Exemples de pI :
- Glycine : pKₐ₁ = 2,34, pKₐ₂ = 9,60 → pI = (2,34 + 9,60)/2 = 5,97
- Acide glutamique : pKₐ₁ = 2,19, pKₐ₂ = 9,67, pKₐ(R) = 4,25 → pI = (pKₐ₁ + pKₐ(R))/2 ≈ 3,22
- Lysine : pKₐ₁ = 2,18, pKₐ₂ = 8,95, pKₐ(R) = 10,53 → pI = (pKₐ₂ + pKₐ(R))/2 ≈ 9,74

🔗 IV – LIAISON PEPTIDIQUE

Formation

La liaison peptidique se forme par réaction entre le groupe -COOH d'un acide aminé et le groupe -NH₂ d'un autre acide aminé, avec élimination d'une molécule d'eau.

H₂N-CH(R₁)-COOH + H₂N-CH(R₂)-COOH → H₂N-CH(R₁)-CO-NH-CH(R₂)-COOH + H₂O

Caractéristiques

  • La liaison peptidique -CO-NH- est partiellement double (caractère plan et rigide)
  • Elle ne peut pas tourner librement autour de l'axe C-N
  • L'enchaînement d'acides aminés par liaisons peptidiques forme les polypeptides et les protéines
📌 Structure des protéines :
- Structure primaire : séquence linéaire des acides aminés
- Structure secondaire : hélice α, feuillet β (liaisons hydrogène)
- Structure tertiaire : repliement tridimensionnel
- Structure quaternaire : association de plusieurs sous-unités

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Un acide α-aminé possède un groupe -NH₂ et un groupe -COOH sur le carbone α
- Le carbone α est asymétrique (sauf pour la glycine)
- Les acides aminés naturels sont de configuration L
- Ce sont des ampholytes (comportement acide et basique)
- À l'état zwitterionique : H₃N⁺-CH(R)-COO⁻
- Le point isoélectrique (pI) est le pH où la charge nette est nulle
- Les acides aminés s'enchaînent par liaisons peptidiques pour former les protéines
🎯 Méthode pour calculer le pI :
1. Identifier les pKₐ du couple acide/base concerné
2. Pour un acide aminé neutre : pI = (pK_{a1} + pK_{a2})/2
3. Pour un acide aminé acide (R acide) : pI = (pK_{a1} + pK_{a(R)})/2
4. Pour un acide aminé basique (R basique) : pI = (pK_{a2} + pK_{a(R)})/2
