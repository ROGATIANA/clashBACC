Acide fort - Base forte

Dissociation totale · Calcul du pH · Dosage acido-basique · Courbe de titrage · Équivalence

Cours complet

⚡ INTRODUCTION

Un acide fort est un acide qui se dissocie totalement dans l'eau. Une base forte est une base qui se dissocie totalement dans l'eau. La réaction entre un acide fort et une base forte est une réaction quasi-totale.

📌 Exemples :
- Acides forts : HCl, HBr, HI, HNO₃, H₂SO₄ (1ère acidité), HClO₄
- Bases fortes : NaOH, KOH, LiOH, Ba(OH)₂, Ca(OH)₂

📚 I – ACIDES FORTS

Dissociation totale

HA + H₂O → H₃O⁺ + A⁻

ou simplement : HA → H⁺ + A⁻

Calcul du pH

Pour une solution d'acide fort de concentration C_a :

[H₃O⁺] = C_a (si C_a ≫ 10⁻⁷ mol/L)

pH = -log C_a

📌 Remarque : Pour les solutions très diluées (C_a < 10⁻⁶ mol/L), il faut tenir compte de l'autoprotolyse de l'eau.

📚 II – BASES FORTES

Dissociation totale

B + H₂O → BH⁺ + OH⁻
ou pour les hydroxydes : NaOH → Na⁺ + OH⁻

Calcul du pH

Pour une solution de base forte de concentration C_b :

[OH⁻] = C_b (si C_b ≫ 10⁻⁷ mol/L)

pOH = -log C_b et pH = 14 - pOH

Soit : pH = 14 + log C_b

⚗️ III – RÉACTION ACIDE FORT - BASE FORTE

Équation bilan

H₃O⁺ + OH⁻ → 2H₂O

Cette réaction est quasi-totale (constante de réaction K_r = 1/Kₑ = 10¹⁴).

Calcul du pH après mélange

Soit un mélange de V_a mL d'acide fort à C_a mol/L et V_b mL de base forte à C_b mol/L :

  • Cas 1 : C_a·V_a > C_b·V_b → acide en excès → pH = -log(C_a·V_a - C_b·V_b)/(V_a+V_b)
  • Cas 2 : C_a·V_a = C_b·V_b → équivalence → pH = 7 (à 25°C)
  • Cas 3 : C_a·V_a < C_b·V_b → base en excès → pH = 14 + log(C_b·V_b - C_a·V_a)/(V_a+V_b)

📊 IV – DOSAGE ACIDE FORT - BASE FORTE

Principe

On verse progressivement une base forte dans un acide fort (ou l'inverse). Le pH est mesuré en fonction du volume de solution titrante versé.

Point d'équivalence

Au point d'équivalence, les quantités d'acide et de base sont stoechiométriques :

C_a·V_a = C_b·V_b

À l'équivalence : pH = 7 (solution neutre)

Saut de pH

Autour du point d'équivalence, on observe un saut de pH très important. Ce saut permet de repérer l'équivalence avec un indicateur coloré dont la zone de virage est dans cet intervalle (hélianthine, phénolphtaléine ou bleu de bromothymol).

📌 Avantages :
- Réaction simple et rapide
- Saut de pH très marqué (facilité de repérage)
- De nombreux indicateurs conviennent

📝 V – FORMULAIRE RÉCAPITULATIF

Acide fort (concentration C)

pH = -log C

Base forte (concentration C)

pH = 14 + log C

À l'équivalence

C_a·V_a = C_b·V_b
pH = 7

Acide en excès

pH = -log[(n_a - n_b)/(V_a+V_b)]

Base en excès

pH = 14 + log[(n_b - n_a)/(V_a+V_b)]

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Les acides forts et les bases fortes sont totalement dissociés dans l'eau
- pH d'un acide fort : pH = -log C
- pH d'une base forte : pH = 14 + log C
- La réaction H₃O⁺ + OH⁻ → 2H₂O est quasi-totale
- Au point d'équivalence : pH = 7 et C_a·V_a = C_b·V_b
- Le saut de pH autour de l'équivalence permet un repérage facile
🎯 Méthode pour un mélange acide fort/base forte :
1. Calculer les quantités de matière : n_a = C_a·V_a, n_b = C_b·V_b
2. Déterminer le réactif limitant
3. Si n_a > n_b : excès d'acide → pH = -log[(n_a - n_b)/V_total]
4. Si n_a = n_b : équivalence → pH = 7
5. Si n_a < n_b : excès de base → pH = 14 + log[(n_b - n_a)/V_total]
