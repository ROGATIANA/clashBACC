📚 PARTIE 1 : NOMBRES COMPLEXES

📌 Formes et opérations

Forme algébrique

z = a + ib
a, b ∈ ℝ, i² = -1

Module

|z| = √(a² + b²)

Argument

θ = arg(z)
tan θ = b/a

Forme trigonométrique

z = r(cos θ + i sin θ)

Forme exponentielle

z = re^{iθ}

Conjugué

\bar{z} = a - ib
|z|² = z·\bar{z}

📌 Formules de Moivre et Euler

Formule de Moivre

(cos θ + i sin θ)^n = cos(nθ) + i sin(nθ)

Formules d'Euler

cos θ = (e^{iθ} + e^{-iθ})/2
sin θ = (e^{iθ} - e^{-iθ})/(2i)

📈 PARTIE 2 : LIMITES ET CONTINUITÉ

Limites usuelles

lim_{x→0} sin x / x = 1
lim_{x→0} (e^x - 1)/x = 1
lim_{x→+∞} ln x / x = 0
lim_{x→+∞} e^x / x = +∞

Théorème des gendarmes

Si u_n ≤ v_n ≤ w_n
et lim u_n = lim w_n = L
alors lim v_n = L

Théorème des valeurs intermédiaires

Si f continue sur [a,b]
et k entre f(a) et f(b)
alors ∃ c ∈ [a,b] : f(c)=k

⚡ PARTIE 3 : DÉRIVATION

📌 Dérivées usuelles

Fonction puissance

(x^n)' = n·x^{n-1}

Exponentielle

(e^x)' = e^x
(e^{u})' = u'·e^{u}

Logarithme

(ln x)' = 1/x
(ln u)' = u'/u

Trigonométriques

(sin x)' = cos x
(cos x)' = -sin x
(tan x)' = 1/cos²x = 1 + tan²x

📌 Règles de dérivation

Somme et produit

(u+v)' = u' + v'
(uv)' = u'v + uv'

Quotient

(u/v)' = (u'v - uv')/v²

Fonction composée

(f∘g)' = (f'∘g)·g'

🔢 PARTIE 4 : SUITES NUMÉRIQUES

Suite arithmétique

u_{n+1} = u_n + r
u_n = u_0 + nr
S_n = (n+1)(u_0+u_n)/2

Suite géométrique

u_{n+1} = q·u_n
u_n = u_0·q^n
S_n = u_0·(1-q^{n+1})/(1-q) (q≠1)

∫ PARTIE 5 : CALCUL INTÉGRAL

📌 Intégrales usuelles

Fonction puissance

∫ x^n dx = x^{n+1}/(n+1) + C (n≠-1)

Inverse

∫ 1/x dx = ln|x| + C

Exponentielle

∫ e^x dx = e^x + C

Trigonométriques

∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C

📌 Règles d'intégration

Intégration par parties

∫ u'v = uv - ∫ uv'

Théorème fondamental

∫_a^b f'(x)dx = f(b) - f(a)

Valeur moyenne

μ = 1/(b-a) ∫_a^b f(x)dx

🎲 PARTIE 6 : PROBABILITÉS

Probabilité conditionnelle

P(A|B) = P(A∩B)/P(B)

Formule des probabilités totales

P(B) = Σ P(A_i)·P(B|A_i)

Formule de Bayes

P(A|B) = P(A)·P(B|A)/P(B)

📌 Lois usuelles

Loi binomiale B(n,p)

P(X=k) = C_n^k p^k (1-p)^{n-k}
E(X)=np, V(X)=np(1-p)

Loi de Poisson P(λ)

P(X=k) = e^{-λ}·λ^k/k!
E(X)=λ, V(X)=λ

Loi normale N(μ,σ²)

f(x) = 1/(σ√(2π))·e^{-(x-μ)²/(2σ²)}

📐 PARTIE 7 : ÉQUATIONS DIFFÉRENTIELLES

Ordre 1 (sans second membre)

y' + a(x)y = 0
y(x) = C·e^{-∫a(x)dx}

Ordre 1 (avec second membre)

y' + a(x)y = b(x)
Solution = SH + SP

Ordre 2 (coefficients constants)

y'' + py' + qy = 0
Équation caractéristique : r² + pr + q = 0

📊 PARTIE 8 : FONCTIONS LOGARITHME ET EXPONENTIELLE

Propriétés du logarithme

ln(ab) = ln a + ln b
ln(a/b) = ln a - ln b
ln(a^n) = n·ln a
ln(1) = 0, ln(e) = 1

Propriétés de l'exponentielle

e^{a+b} = e^a·e^b
e^{a-b} = e^a/e^b
(e^a)^n = e^{na}
e^0 = 1, e^1 = e

📌 CONSTANTES FONDAMENTALES

Nombre d'Euler

e ≈ 2,71828

Pi

π ≈ 3,14159

Nombre d'or

φ = (1+√5)/2 ≈ 1,618
