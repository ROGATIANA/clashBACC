Équations différentielles

Équations du 1er ordre · Équations du 2nd ordre · Solutions générales · Conditions initiales

Cours complet

📐 INTRODUCTION

Une équation différentielle est une équation qui établit une relation entre une fonction inconnue et ses dérivées successives. Elle est fondamentale en physique, en biologie et en économie pour modéliser des phénomènes d'évolution (croissance, décroissance, mouvements oscillatoires, etc.).

📌 Exemples :
- y' = ky (croissance exponentielle)
- y'' + ω²y = 0 (oscillateur harmonique)
- y' + ay = b (modèle avec forçage)

📚 I – L'ÉQUATION y' + ay = 0 (a réel)

Fig. 1 – Équation différentielle linéaire du 1er ordre sans second membre
📌 Solution générale :
Les solutions de y' + ay = 0 sont de la forme :
y(x) = C·e^{-ax} où C ∈ ℝ (constante réelle)
📌 Remarque :
Cette équation modélise des phénomènes de décroissance exponentielle (a > 0) ou de croissance exponentielle (a < 0).

📐 II – L'ÉQUATION y'' + ay' + by = 0 (a, b des réels)

Fig. 2 – Équation différentielle linéaire du 2nd ordre à coefficients constants
📌 Équation caractéristique :
On associe l'équation r² + a r + b = 0. La solution dépend du discriminant Δ = a² - 4b.

📊 III – LES RÉSULTATS

Fig. 3 – Tableau récapitulatif des solutions selon le discriminant
📌 Cas du discriminant Δ > 0 (deux racines réelles distinctes) :
y(x) = C₁·e^{r₁x} + C₂·e^{r₂x}
📌 Cas du discriminant Δ = 0 (racine double) :
y(x) = (C₁ + C₂·x)·e^{r₀x}
📌 Cas du discriminant Δ < 0 (racines complexes conjuguées r = α ± iβ) :
y(x) = e^{αx}(C₁·cos(βx) + C₂·sin(βx))

🎯 IV – AVEC LES CONDITIONS INITIALES

Fig. 4 – Détermination des constantes à l'aide des conditions initiales
📌 Principe :
Les conditions initiales permettent de déterminer les constantes C₁ et C₂. On utilise généralement :
- y(x₀) = y₀ (valeur de la fonction en un point)
- y'(x₀) = y'₀ (valeur de la dérivée en un point)

⚡ V – ÉQUATION AVEC SECOND MEMBRE

Fig. 5 – Équation différentielle avec second membre
📌 Principe de superposition :
La solution générale d'une équation avec second membre est la somme :
y = y₀ + y_p
où :
  • y₀ : solution générale de l'équation sans second membre (solution homogène)
  • y_p : solution particulière de l'équation avec second membre
📌 Recherche d'une solution particulière :
- Si le second membre est un polynôme : chercher y_p polynôme de même degré
- Si le second membre est une exponentielle : chercher y_p de la forme k·e^{λx}
- Si le second membre est une fonction trigonométrique : chercher y_p de la forme A·cos(ωx) + B·sin(ωx)

📝 FORMULAIRE RÉCAPITULATIF

1er ordre homogène

y' + ay = 0
y = C·e^{-ax}

1er ordre avec second membre

y' + ay = b
y = C·e^{-ax} + b/a (a ≠ 0)

2nd ordre Δ > 0

y = C₁e^{r₁x} + C₂e^{r₂x}

2nd ordre Δ = 0

y = (C₁ + C₂x)e^{r₀x}

2nd ordre Δ < 0

y = e^{αx}(C₁cos βx + C₂ sin βx)

Principe superposition

y = y₀ + y_p

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- 1er ordre homogène : y' + ay = 0 → y = C·e^{-ax}
- 2nd ordre homogène : résoudre l'équation caractéristique r² + ar + b = 0
- Δ > 0 : deux exponentielles réelles
- Δ = 0 : exponentielle × polynôme de degré 1
- Δ < 0 : oscillatoire amorti e^{αx}(C₁cos βx + C₂ sin βx)
- Avec second membre : solution = solution homogène + solution particulière
- Conditions initiales : permettent de trouver les constantes C₁, C₂
🎯 Méthode pour résoudre une équation différentielle :
1. Identifier le type d'équation (1er ordre, 2nd ordre, avec/sans second membre)
2. Résoudre l'équation homogène (sans second membre)
3. Trouver une solution particulière (si second membre présent)
4. Additionner solution homogène + solution particulière
5. Utiliser les conditions initiales pour déterminer les constantes
