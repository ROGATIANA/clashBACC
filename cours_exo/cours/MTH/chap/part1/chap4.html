🎲 INTRODUCTION

Les probabilités sont une branche des mathématiques qui étudie les phénomènes aléatoires. Elles permettent de quantifier l'incertitude et de modéliser des situations où le résultat n'est pas connu à l'avance (lancers de dés, tirages au sort, etc.). La théorie des probabilités est essentielle en statistiques, en physique, en finance et dans de nombreux domaines scientifiques.

📌 Vocabulaire de base :
- Expérience aléatoire : expérience dont le résultat dépend du hasard
- Univers Ω : ensemble de tous les résultats possibles
- Événement : sous-ensemble de Ω
- Probabilité P : mesure sur l'ensemble des événements

📌 Définition (Laplace) :
En situation d'équiprobabilité, la probabilité d'un événement A est :
P(A) = \text{nombre de cas favorables} / \text{nombre de cas possibles}
📌 Probabilité conditionnelle :
P(A|B) = P(A∩B)/P(B) pour P(B) > 0
P(A∩B) = P(A|B)·P(B) = P(B|A)·P(A)
📌 Formule des probabilités totales :
Soit {B₁, B₂, ..., B_n} une partition de Ω, alors :
P(A) = Σ P(A|B_i)·P(B_i)
📌 Formule de Bayes :
P(B_i|A) = \frac{P(A|B_i)·P(B_i)}{Σ P(A|B_j)·P(B_j)}
📌 Indépendance :
Deux événements A et B sont indépendants si :
P(A∩B) = P(A)·P(B)
Cela équivaut à P(A|B) = P(A) (ou P(B|A) = P(B))

📝 FORMULAIRE RÉCAPITULATIF

Combinaisons

C_n^k = n!/(k!(n-k)!)

Probabilité conditionnelle

P(A|B) = P(A∩B)/P(B)

Formule de Bayes

P(B|A) = P(A|B)·P(B)/P(A)

Indépendance

P(A∩B) = P(A)·P(B)

Probabilité de l'union

P(A∪B) = P(A) + P(B) - P(A∩B)

Événement contraire

P(\bar{A}) = 1 - P(A)

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Dénombrement : maîtriser factorielles, arrangements, combinaisons
- Probabilité : P(A) = cas favorables/cas possibles (équiprobabilité)
- Union : P(A∪B) = P(A) + P(B) - P(A∩B)
- Conditionnement : P(A∩B) = P(A|B)·P(B)
- Formule des probabilités totales : très utile pour les problèmes à plusieurs étapes
- Formule de Bayes : permet de "retourner" le conditionnement
- Indépendance : P(A∩B) = P(A)·P(B)
🎯 Méthode pour résoudre un problème de probabilités :
1. Définir l'univers Ω et les événements concernés
2. Déterminer si la situation est équiprobable
3. Utiliser les formules de dénombrement si nécessaire
4. Appliquer les règles de calcul des probabilités (union, intersection, conditionnement)
5. Pour les problèmes à plusieurs étapes : construire un arbre de probabilités
