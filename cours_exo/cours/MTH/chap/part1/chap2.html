Suites numériques

Généralités · Limites · Convergence · Suites arithmétiques et géométriques

Cours complet

🔢 INTRODUCTION

Une suite numérique est une fonction définie sur ℕ (ou une partie de ℕ) à valeurs dans ℝ. Elle peut être définie de manière explicite (u_n = f(n)) ou par récurrence (u_{n+1} = f(u_n)). Les suites sont fondamentales en mathématiques pour modéliser des phénomènes discrets (croissance, décroissance, intérêts, etc.).

📌 Exemples :
- Suite arithmétique : u_{n+1} = u_n + r
- Suite géométrique : u_{n+1} = q·u_n
- Suite définie par récurrence : u_{n+1} = f(u_n)

📌 Suites particulières :
- Suite arithmétique : u_n = u_0 + nr
- Suite géométrique : u_n = u_0·q^n
- Suite majorée : ∃ M ∈ ℝ tel que u_n ≤ M pour tout n
- Suite minorée : ∃ m ∈ ℝ tel que u_n ≥ m pour tout n
- Suite bornée : suite à la fois majorée et minorée

📌 Théorème de convergence monotone :
- Une suite croissante et majorée converge vers sa borne supérieure
- Une suite décroissante et minorée converge vers sa borne inférieure
- Une suite monotone et non bornée diverge vers ±∞
📌 Théorème des gendarmes :
Si v_n ≤ u_n ≤ w_n à partir d'un certain rang, et lim v_n = lim w_n = L, alors lim u_n = L.

🔬 III – COMPLÉMENT SUR LES SUITES

Fig. 10 – Suites adjacentes

📌 Suites adjacentes :
Deux suites (u_n) et (v_n) sont dites adjacentes si :
- (u_n) est croissante et (v_n) est décroissante
- lim (v_n - u_n) = 0
Alors les deux suites convergent vers la même limite.
📌 Suites récurrentes :
Pour une suite définie par u_{n+1} = f(u_n) :
- Si f est continue et la suite converge vers L, alors L = f(L) (point fixe)
- La limite est solution de l'équation f(x) = x

📝 FORMULAIRE RÉCAPITULATIF

Suite arithmétique

u_n = u_0 + nr
S_n = (n+1)(u_0+u_n)/2

Suite géométrique

u_n = u_0·q^n
S_n = u_0(1-q^{n+1})/(1-q) (q≠1)

Somme des n premiers entiers

1 + 2 + ... + n = n(n+1)/2

Somme des n premiers carrés

1² + 2² + ... + n² = n(n+1)(2n+1)/6

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Suite arithmétique : variation additive constante (r = u_{n+1} - u_n)
- Suite géométrique : variation multiplicative constante (q = u_{n+1}/u_n)
- Convergence : la limite est finie
- Divergence : la limite est ±∞ ou n'existe pas
- Théorème de convergence monotone : croissante+majorée converge ; décroissante+minorée converge
- Théorème des gendarmes : encadrement pour prouver une limite
- Les suites adjacentes convergent vers la même limite
🎯 Méthode pour étudier une suite :
1. Déterminer si la suite est définie explicitement ou par récurrence
2. Étudier son sens de variation (u_{n+1} - u_n ou u_{n+1}/u_n)
3. Chercher une éventuelle limite (majoration, minoration)
4. Appliquer le théorème de convergence monotone si applicable
5. Pour une suite récurrente, trouver la limite en résolvant L = f(L)
