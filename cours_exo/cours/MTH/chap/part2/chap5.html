Fonctions ln et exp

Logarithme népérien · Exponentielle · Propriétés · Dérivées · Limites · Puissances

Cours complet

📈 INTRODUCTION

Les fonctions logarithme népérien (ln) et exponentielle (exp) sont des fonctions réciproques l'une de l'autre. Elles sont essentielles en mathématiques pour modéliser des phénomènes de croissance (populations, intérêts composés) ou de décroissance (radioactivité, refroidissement).

📌 Relation fondamentale :
Pour tout x ∈ ℝ, ln(e^x) = x
Pour tout x > 0, e^{ln x} = x

📐 I – FONCTION LOGARITHME NÉPÉRIEN

Fig. 1 – Définition, propriétés, dérivée, limites et courbe de ln(x)
📌 Définition :
La fonction logarithme népérien, notée ln, est définie sur ]0, +∞[ par :
ln(1) = 0 et ln'(x) = 1/x
📌 Propriétés algébriques (pour a,b > 0, n ∈ ℝ) :
- ln(ab) = ln a + ln b
- ln(a/b) = ln a - ln b
- ln(a^n) = n·ln a
- ln(√a) = ½·ln a
📌 Dérivée :
- (ln x)' = 1/x
- (ln|u|)' = u'/u (pour u ≠ 0)
📌 Limites :
- lim_{x→0⁺} ln x = -∞
- lim_{x→+∞} ln x = +∞
- lim_{x→0⁺} x·ln x = 0
- lim_{x→+∞} (ln x)/x = 0

⚡ II – FONCTION EXPONENTIELLE

Fig. 2 – Définition, propriétés, dérivée, limites et courbe de exp(x)
📌 Définition :
La fonction exponentielle, notée exp(x) = e^x, est définie sur ℝ comme la réciproque de ln.
exp(0) = 1 et (e^x)' = e^x
📌 Propriétés algébriques (pour a,b ∈ ℝ) :
- e^{a+b} = e^a·e^b
- e^{a-b} = e^a/e^b
- (e^a)^n = e^{na}
- e^0 = 1, e^1 = e
📌 Dérivée :
- (e^x)' = e^x
- (e^{u})' = u'·e^{u}
📌 Limites :
- lim_{x→-∞} e^x = 0
- lim_{x→+∞} e^x = +∞
- lim_{x→+∞} e^x/x = +∞
- lim_{x→-∞} x·e^x = 0

📊 III – PUISSANCE D'UN NOMBRE POSITIF

Fig. 3 – Définition de a^α pour a > 0 et α réel
📌 Définition :
Pour a > 0 et α ∈ ℝ, on définit :
a^α = e^{α·ln a}
📌 Propriétés (pour a,b > 0, α,β ∈ ℝ) :
- a^α·a^β = a^{α+β}
- a^α/a^β = a^{α-β}
- (a^α)^β = a^{αβ}
- (ab)^α = a^α·b^α
- (a/b)^α = a^α/b^α
📌 Dérivée de x^α :
(x^α)' = α·x^{α-1} pour x > 0
📌 Dérivée de a^x (a > 0) :
(a^x)' = a^x·ln a

📐 IV – ÉQUATIONS ET INÉQUATIONS

📌 Résolution d'équations avec ln et exp :
- ln A = ln B ⇔ A = B > 0
- e^{A} = e^{B} ⇔ A = B
- ln A = B ⇔ A = e^{B} (avec A > 0)
- e^{A} = B ⇔ A = ln B (avec B > 0)
📌 Inégalités :
- ln x < ln y ⇔ 0 < x < y
- e^x < e^y ⇔ x < y
(car les fonctions ln et exp sont strictement croissantes)

📈 V – CROISSANCES COMPARÉES

📌 À retenir (en +∞) :
e^x ≫ x^n ≫ ln x
L'exponentielle l'emporte sur toute puissance, qui l'emporte sur le logarithme.

En +∞

lim e^x/x^n = +∞

En +∞

lim x^n/ln x = +∞

En 0⁺

lim x·ln x = 0

En -∞

lim x·e^x = 0

📝 FORMULAIRE RÉCAPITULATIF

Dérivée de ln

(ln x)' = 1/x

Dérivée de e^x

(e^x)' = e^x

Propriétés de ln

ln(ab) = ln a + ln b
ln(a^n) = n·ln a

Propriétés de exp

e^{a+b} = e^a·e^b
e^{na} = (e^a)^n

Dérivée de a^x

(a^x)' = a^x·ln a

Dérivée de ln|u|

(ln|u|)' = u'/u

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Domaine de ln : ]0, +∞[ (logarithme d'un nombre négatif ou nul : impossible)
- Domaine de exp : ℝ (toujours définie)
- Propriétés fondamentales : transformer produits en sommes, puissances en produits
- Dérivée de ln u : u'/u (nécessite u > 0)
- Dérivée de e^u : u'·e^u
- Croissances comparées : en +∞, e^x domine x^n, qui domine ln x
- Fonctions réciproques : ln et exp sont réciproques (courbes symétriques par rapport à y=x)
🎯 Méthode pour résoudre une équation avec ln/exp :
1. Déterminer le domaine (pour ln : argument > 0)
2. Isoler le logarithme ou l'exponentielle
3. Appliquer la fonction réciproque (exp ou ln)
4. Résoudre l'équation obtenue
5. Vérifier les solutions dans le domaine
