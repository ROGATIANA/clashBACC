Cours complet

📊 INTRODUCTION

L'étude complète d'une fonction permet de tracer sa courbe représentative et de comprendre son comportement sur son domaine de définition. Elle nécessite l'utilisation de tous les outils de l'analyse : limites, dérivées, asymptotes, etc.

📌 Objectifs :
- Déterminer le domaine de définition
- Étudier les limites et les asymptotes
- Calculer la dérivée et étudier les variations
- Trouver les extremums et les points d'inflexion
- Tracer la courbe représentative

📚 I – GÉNÉRALITÉS SUR LES FONCTIONS

Fig. 1 – Domaines de définition, parité, périodicité
Fig. 2 – Fonctions paires, impaires et leurs propriétés graphiques
Fig. 3 – Fonctions périodiques et leur étude
📌 Fonction paire :
f(-x) = f(x) pour tout x ∈ D
La courbe est symétrique par rapport à l'axe des ordonnées (Oy).
Étude sur [0, +∞[ suffit.
📌 Fonction impaire :
f(-x) = -f(x) pour tout x ∈ D
La courbe est symétrique par rapport à l'origine O(0,0).
Étude sur [0, +∞[ suffit.
📌 Fonction périodique de période T :
f(x + T) = f(x) pour tout x ∈ D
Étude sur un intervalle de longueur T suffit.

📐 II – PLAN D'ÉTUDE D'UNE FONCTION

Fig. 4 – Méthodologie complète pour l'étude d'une fonction
📌 Étapes de l'étude :
1. Déterminer le domaine de définition D_f
2. Étudier la parité et la périodicité (pour réduire l'intervalle d'étude)
3. Calculer les limites aux bornes du domaine et aux points singuliers
4. Déterminer les asymptotes (verticales, horizontales, obliques)
5. Calculer la dérivée f' et étudier son signe
6. Dresser le tableau de variations (avec limites et extremums)
7. Calculer la dérivée seconde f'' (convexité, points d'inflexion)
8. Tracer la courbe représentative (points particuliers, tangentes)
📌 Asymptotes :
- Asymptote verticale : x = a si lim_{x→a} f(x) = ±∞
- Asymptote horizontale : y = L si lim_{x→±∞} f(x) = L
- Asymptote oblique : y = ax + b avec a = lim f(x)/x et b = lim (f(x) - ax)

🔄 III – FONCTION RÉCIPROQUE

Fig. 5 – Théorème de la bijection et fonction réciproque
📌 Théorème de la bijection :
Si f est continue et strictement monotone sur un intervalle I, alors f réalise une bijection de I sur f(I).
La fonction réciproque f^{-1} est définie sur f(I).
📌 Propriétés de la fonction réciproque :
- f^{-1}(f(x)) = x pour tout x ∈ I
- f(f^{-1}(y)) = y pour tout y ∈ J
- Les courbes de f et f^{-1} sont symétriques par rapport à la droite y = x
- (f^{-1})'(y_0) = 1 / f'(x_0) où y_0 = f(x_0)
📌 Exemples de fonctions réciproques :
- f(x) = ln x sur ]0, +∞[ → f^{-1}(x) = e^x
- f(x) = sin x sur [-π/2, π/2] → f^{-1}(x) = arcsin x
- f(x) = cos x sur [0, π] → f^{-1}(x) = arccos x
- f(x) = tan x sur ]-π/2, π/2[ → f^{-1}(x) = arctan x

📝 FORMULAIRE RÉCAPITULATIF

Symétrie par rapport à Oy

Fonction impaire

f(-x) = -f(x)
Symétrie par rapport à O

Fonction périodique

f(x + T) = f(x)
Étude sur [0,T]

Asymptote horizontale

lim f(x) = L → y = L

Asymptote verticale

lim f(x) = ±∞ → x = a

Dérivée de la réciproque

(f^{-1})'(y) = 1/f'(x)

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Domaine de définition : première étape indispensable (valeur interdite = dénominateur nul, log d'argument ≤ 0, racine d'argument négatif)
- Parité/Périodicité : permet de réduire l'intervalle d'étude
- Limites : étudier aux bornes du domaine et aux points singuliers
- Asymptotes : verticales (dénominateur nul), horizontales (limite finie en ±∞), obliques (limite infinie)
- Tableau de variations : résume les limites, le signe de f' et les variations
- Fonction réciproque : existe si f est bijective (continue et strictement monotone)
🎯 Méthode pour tracer une courbe :
1. Déterminer le domaine et réduire si parité/périodicité
2. Tracer les asymptotes en pointillés
3. Placer les points particuliers (extremums, intersections avec les axes)
4. Tracer la courbe en respectant les variations et les asymptotes
5. Vérifier la cohérence avec les propriétés (parité, limites, etc.)
