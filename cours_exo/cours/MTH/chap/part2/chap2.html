Cours complet

⚡ INTRODUCTION

La dérivation est l'opération qui permet de calculer le taux de variation instantané d'une fonction. Elle est essentielle en analyse pour étudier les variations, les extremums, la convexité, etc. La primitive est l'opération inverse de la dérivation : trouver une fonction F telle que F' = f.

📌 Lien dérivation / primitive :
- Dériver : f → f'
- Primitiver : f → F (avec F' = f)
- Si F est une primitive de f, alors toutes les primitives sont de la forme F + C (C ∈ ℝ)

📌 Définition de la dérivée :
f'(a) = lim_{h→0} [f(a+h) - f(a)]/h
(si cette limite existe et est finie)
📌 Règles de dérivation :
- (u + v)' = u' + v'
- (uv)' = u'v + uv'
- (1/u)' = -u'/u²
- (u/v)' = (u'v - uv')/v²
- (f∘g)' = (f'∘g)·g' (dérivée d'une composée)

📚 II – PRIMITIVES

Fig. 5 – Tableau des primitives usuelles
📌 Définition :
Une primitive d'une fonction f sur un intervalle I est une fonction F dérivable sur I telle que :
F'(x) = f(x) pour tout x ∈ I.
📌 Primitives usuelles :
- ∫ x^n dx = x^{n+1}/(n+1) + C (n ≠ -1)
- ∫ 1/x dx = ln|x| + C
- ∫ e^x dx = e^x + C
- ∫ sin x dx = -cos x + C
- ∫ cos x dx = sin x + C
- ∫ 1/cos²x dx = tan x + C
- ∫ 1/(1 + x²) dx = arctan x + C
📌 Propriétés des primitives :
- La primitive d'une somme est la somme des primitives
- Si F est une primitive de f, alors toutes les primitives sont de la forme F + C (C constante réelle)
- Une primitive d'une fonction de la forme u'·(u^n) est u^{n+1}/(n+1)
- Une primitive d'une fonction de la forme u'/u est ln|u|

📝 FORMULAIRE RÉCAPITULATIF

Dérivée de x^n

(x^n)' = n·x^{n-1}

Dérivée de e^x

(e^x)' = e^x

Dérivée de ln x

(ln x)' = 1/x

Dérivée de sin x

(sin x)' = cos x

Dérivée de cos x

(cos x)' = -sin x

Dérivée d'une composée

(f∘g)' = (f'∘g)·g'

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- Dérivée : taux de variation instantané, pente de la tangente
- Signe de la dérivée : f' > 0 → f croissante ; f' < 0 → f décroissante
- Extremums : si f' s'annule en changeant de signe → extremum local
- Primitive : opération inverse de la dérivation (à une constante près)
- Une primitive n'est jamais unique (famille à une constante additive près)
- Une fonction continue sur un intervalle admet toujours des primitives
🎯 Méthode pour dériver une fonction :
1. Identifier le type de fonction (puissance, exponentielle, logarithme, trigo, composée)
2. Appliquer la règle de dérivation adaptée
3. Simplifier l'expression obtenue
4. Étudier le signe de la dérivée pour les variations
🎯 Méthode pour calculer une primitive :
1. Reconnaître la forme de la fonction (tableau des primitives usuelles)
2. Identifier la fonction dont la dérivée correspond (remonter le tableau de dérivation)
3. Pour les fonctions composées du type u'·f'(u) : primitive = f(u) + C
4. Ne pas oublier la constante d'intégration C
