Oscillations mécaniques libres

Système masse-ressort · Équation différentielle · Période · Énergie mécanique

Cours complet

🔄 INTRODUCTION

Un oscillateur mécanique est un système qui effectue un mouvement de va-et-vient autour d'une position d'équilibre. Les oscillations sont dites libres lorsqu'elles s'effectuent sans entretien extérieur, sous l'action des forces internes (rappel) et éventuellement des frottements.

📌 Systèmes étudiés :
- Système masse-ressort horizontal
- Système masse-ressort vertical
- Pendule élastique
- Oscillations amorties

📏 I – SYSTÈME MASSE-RESSORT HORIZONTAL

Mise en équation (PFD)

Soit une masse m attachée à un ressort de raideur k, posée sur un support horizontal sans frottement. La position x est repérée par rapport à la position d'équilibre.

Force de rappel : F = -k·x (loi de Hooke)

Application du PFD : m·x'' = -k·x

Équation différentielle : x'' + (k/m)·x = 0

Soit x'' + ω₀²·x = 0 avec ω₀² = k/m

Solution de l'équation différentielle

x(t) = X_m·cos(ω₀·t + φ)

où :

  • X_m : amplitude (m)
  • ω₀ = √(k/m) : pulsation propre (rad/s)
  • φ : phase à l'origine (rad)

Période propre

T₀ = 2π/ω₀ = 2π·√(m/k)

La période est indépendante de l'amplitude (isochronisme).

⬇️ II – SYSTÈME MASSE-RESSORT VERTICAL

Position d'équilibre

À l'équilibre : m·g = k·Δℓ₀ → Δℓ₀ = m·g/k

Mise en équation

En prenant l'origine à la position d'équilibre :

m·z'' = -k·z

Soit z'' + (k/m)·z = 0 (même équation qu'horizontal)

Même pulsation et même période : T₀ = 2π·√(m/k)

📌 Remarque : La pesanteur n'influence pas la période des oscillations, elle modifie seulement la position d'équilibre.

⚡ III – ÉNERGIE MÉCANIQUE DE L'OSCILLATEUR

Énergie potentielle élastique

E_pe = ½·k·x²

Énergie cinétique

E_c = ½·m·v² = ½·m·(x')²

Énergie mécanique (sans frottement)

E_m = E_c + E_pe = ½·k·X_m² = constante

L'énergie mécanique se conserve : elle passe de la forme potentielle à la forme cinétique et inversement.

📌 Expression en fonction du temps :
- E_c(t) = ½·k·X_m²·sin²(ω₀t+φ)
- E_pe(t) = ½·k·X_m²·cos²(ω₀t+φ)

📉 IV – OSCILLATIONS LIBRES AMORTIES

Avec frottement fluide

Force de frottement : F_f = -λ·v (λ : coefficient de frottement)

Équation différentielle : m·x'' + λ·x' + k·x = 0

Soit x'' + (λ/m)·x' + (k/m)·x = 0

📌 Régimes d'oscillations :
- Régime pseudo-périodique (faible amortissement) : oscillations décroissantes
- Régime apériodique (fort amortissement) : retour à l'équilibre sans oscillation
- Régime critique (amortissement critique) : retour le plus rapide

🔗 V – ASSOCIATION DE RESSORTS

📌 Ressorts en série
1/k_eq = 1/k₁ + 1/k₂
T = 2π·√(m/k_eq)
📌 Ressorts en parallèle
k_eq = k₁ + k₂
T = 2π·√(m/k_eq)

📝 FORMULES À RETENIR

📌 Loi de Hooke
F = -k·x
📌 Équation différentielle
x'' + ω₀²·x = 0
📌 Pulsation propre
ω₀ = √(k/m)
📌 Période propre
T₀ = 2π·√(m/k)
📌 Solution
x(t) = X_m·cos(ω₀t+φ)
📌 Énergie mécanique
E_m = ½·k·X_m²

💡 RÉCAPITULATIF ET CONSEILS

📌 Points clés à retenir :
- L'équation différentielle caractérise le mouvement : x'' + (k/m)·x = 0
- La pulsation propre dépend de k et m : ω₀ = √(k/m)
- La période est indépendante de l'amplitude (isochronisme)
- L'énergie mécanique se conserve dans les oscillations non amorties
- Les frottements provoquent un amortissement (diminution de l'amplitude)
- En série : 1/k_eq = 1/k₁ + 1/k₂ ; en parallèle : k_eq = k₁ + k₂
🎯 Méthode pour les exercices :
1. Identifier le système et les forces
2. Appliquer le PFD pour obtenir l'équation différentielle
3. Identifier ω₀² et en déduire T₀
4. Écrire la solution x(t) avec les conditions initiales
5. Calculer l'énergie mécanique si demandé
